# Contributing to WhatsApp CLI

Thank you for your interest in contributing! We welcome contributions from everyone.

## How to Contribute

1.  **Fork** the repository on GitHub.
2.  **Clone** your fork locally: `git clone https://github.com/Lutzvex/'whatsapp-cli.git`
3.  Create a new **branch** for your feature or bugfix: `git checkout -b my-new-feature`
4.  Make your changes and commit them: `git commit -m "Add some feature"`
5.  **Push** to the branch: `git push origin my-new-feature`
6.  Submit a **Pull Request** on GitHub.

## Code Style

Please ensure your code is formatted using Prettier. Run `npx prettier --write .` before committing.
