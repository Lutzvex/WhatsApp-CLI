## Reporting a Vulnerability

Use this section to tell people how to report a vulnerability.

If you discover a security vulnerability within this project, please send an e-mail to **papertusin56@gmail.com**. All security vulnerabilities will be promptly addressed.

**Please do not report security vulnerabilities through public GitHub issues.**
